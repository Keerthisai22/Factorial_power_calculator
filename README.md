# Factorial and Power Calculator (Python)

A menu-driven Python program that calculates the factorial of a number using recursion and power of a number using a lambda function.

## Features
- Factorial calculation using recursion
- Power calculation using lambda function
- Menu-driven interface
- Runs continuously until exit option is selected

## Technologies Used
- Python 3

## How to Run
1. Download or clone the repository
2. Run the program:
   python factorial_power_calculator.py
